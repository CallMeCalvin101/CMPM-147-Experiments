Experiment 4 - Infinite Worlds

Description

In this experiment, I wanted to create a rippling world, one that will fluctuate over time and shift in color as the user moves around the scene.

Technical

The main rippling effect of the scene is controlled by a noise function. This function will essentially get the noise of a tile in space, and will essentially determine the height of a tile. The higher the tile is, this higher its G value is, causing it to be brighter in color as well.

For the scrolling effect, due to the camera being decoupled from the main function, instead of tracking camera position, simple holding down any arrow key will adjust the color of the drawn tiles, where the horizontal axis will control the red coloring, and the vertical axis controlling the blus coloring. Additionally, instead of looping from 0 to 255, in a cycle, when the value goes below 0, it will turn that decrease into an increase. The same applies when going over 255 as well.
